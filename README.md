## ビューアの操作方法

| 操作 | 動作 |
|---|---|
| ドラッグ | 視点移動 |
| ホイール | ズームイン / アウト |
| ノードクリック | 右パネルに詳細 (説明・キーワード・関連テーマ) |
| 関連テーマクリック | そのノードへジャンプ |
| 上部検索ボックス | 名前 / サブカテゴリ / キーワードでハイライト |
| 左サイドのチェック | カテゴリ / 強度フィルタ |
| リセットボタン | 全表示に戻す |
